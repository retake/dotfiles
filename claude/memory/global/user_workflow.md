Claude Codeは日常のインターフェースではなく「定期メンテナンスツール」として使う。

**Why:** セッション維持が難しく、コンテキスト圧縮で精度が下がる。日常の予定・タスク追加はTodoist/Googleカレンダー直接操作の方が速い。

**How to apply:**
- 週次棚卸し時にセッションを立ち上げ、タスク分割・ラベル整理・プロジェクト再編を行う
- Gmail整理・フィルター追加は必要な時にまとめて実施
- /calコマンドはClaude Code利用時の便利ツールとして使う（日常はGoogleカレンダー直接）
